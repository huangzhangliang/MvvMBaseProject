
#MvvM结合Presenter快速开发架框
网络请求工具：OKGO
图片加载工具：Fresco、Glide
图片选择器：Matisse（知乎开源）
动态权限申请工具：Acp
Banner轮播图：BGABanner
上拉加载、下拉刷新控件：TwinklingRefreshLayout
RecyclerViewAdapter：BGABindingRecyclerViewAdapter
Eventbus：Eventbus3.0
